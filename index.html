Your personal photo archive, stored for free on GitHub.

We'll use GitHub (a trusted service used by millions) to safely store your photos in a private space only you can see. Here's how to set it up — it only takes a minute:

  1. Create a free GitHub account (or sign in if you already have one).

    Go to github.com

  2. Create a private storage space for your photos. Click the link below, then just click the green "Create repository" button — don't change any settings.

    Create "my-photos" storage

  3. Generate a secret key so this app can access your photos. Click the link below, then:

    • Change "Expiration" to "No expiration"
    • Click the green "Generate token" button
    • Copy the token that appears (starts with ghp_)

    Generate your secret key This key stays on your computer and is never sent anywhere except GitHub.
